# Blinkit Analysis Dashboard (Excel)

## Overview

This project presents an interactive Excel dashboard created to analyze the sales performance of a quick-commerce grocery delivery business similar to Blinkit (owned by Zomato).
The dashboard helps understand overall business performance, outlet trends, and product-level insights using dynamic filters.

---

## Business Objective

To analyze retail sales data and provide actionable insights on:

* Overall sales performance
* Outlet size and location impact
* Product category contribution
* Customer behavior trends

---

## Dataset

The dataset is a **sample retail sales dataset** created for learning and portfolio purposes.
It includes information related to sales, item categories, outlet size, outlet location tiers, ratings, and establishment years.

> This is not real-time or proprietary company data.

---

## Dashboard Highlights

* **Key KPIs**: Total Sales, Average Sales, Number of Items, Average Rating
* **Sales trend analysis** by outlet establishment year
* **Item-wise sales analysis** across multiple product categories
* **Outlet performance analysis** by size (Small / Medium / High)
* **Location-based analysis** using Tier 1, Tier 2, and Tier 3 outlets
* **Fat content analysis** (Low Fat vs Regular products)

---

## Interactivity

The dashboard is fully dynamic and allows users to:

* Filter data by outlet size
* Filter data by outlet location tier
* Filter data by item category
* Clear all filters with a single click
* Navigate between dashboard and pivot table sheets

This enables easy slicing and dicing of data to view insights from different perspectives.
